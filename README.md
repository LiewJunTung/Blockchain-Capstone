## Setup:
1. `npm install`
1. Create .mnemonic file and fill in Rinkeby wallet address mnemonic text
1. Deploy via `truffle deploy --network rinkeby --reset`
1. Get the deployment transaction hash of `SolnSquareVerifier` and find out the contract address
1. Update the contract address in `mint.js`
1. Update the index ID in `mint()` function of the `mint.js`, E.G. 1-5
1. Mint token via `node mint.js` for different tokens
